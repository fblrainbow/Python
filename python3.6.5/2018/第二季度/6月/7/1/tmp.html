前面写的教程需要一步步去查看执行的结果，看到很多回复的朋友说碰到很多问题，这次教程是通过Shell脚本直接一次到位，不需要那么多繁琐的操作
工具：putty（登录后台运行脚本）、WinScp（上传文件）
脚本文件：

文件名：oneKeyInstall.sh
#!/bin/bash
apt-get update
apt-get upgrade
apt-get install ubuntu-desktop
ping 127.0.0.1 -c 5
mv 50-ubuntu.conf /usr/share/lightdm/lightdm.conf.d/50-ubuntu.conf
mv 1.profile /root/.profile
ping 127.0.0.1 -c 5
reboot
配置文件-1：1.profile
# ~/.profile: executed by Bourne-compatible login shells.

if [ "$BASH" ]; then
  if [ -f ~/.bashrc ]; then
    . ~/.bashrc
  fi
fi
tty -s && mesg n || true

配置文件-2：50-ubuntu.conf
[Seat:*]
user-session=ubuntu
greeter-show-manual-login=true
allow-guest=false

注意：如果是将这些代码直接写到文件（在win7），一定要保存为unix-ANSI格式，否则win7默认是gbk，复制到linux系统会执行可能会由于编码问题而失败（可以直接下载我的资源文件）

操作步骤：
1、通过WinSCP文件传输工具将这三个文件放在服务器的同一个文件夹下




2、putty登录后台执行脚本
bash oneKeyInstall.sh

3、执行结束后在阿里云终端登录（可以用root登录）

一.win7装ubuntu

工具：软通牒
1.把u盘格式化为FAT-32(默认）
2.安装软碟通(网上找注册码注册以后再使用）
如果没有可用如下注册码(亲测可用)：


户名：王涛
注册码：7C81-1689-4046-626F

3.打开要安装的系统

4.写入硬盘映像

5.如果前面没有格式化的在这里也可以选择格式化，已经格式化过的请忽略。（为了一次成功，建议再格式化一次，因为有次只格式了前面一次后来导致安装不成功），这里写入方式选择USB-HDD+(因为是.iso格式)。

6.写入

7.刻录成功（写完之后最好查看下，事件里面数据，介质，扇区之类的是否有写入错误，如果有错误请重新做启动盘，否则安装时进行到一半会出问题）

8.做好启动盘之后就可以安装了.







二： 注意几点：

1.如果安装不成功百度下bios设置问题，我第一次安装就是bios设置错误导致安装不成功，大部分bios设置是开机F2,一定要按开机键之前就按住F2,否则很可能进不去bios,因为电脑自检过程非常快，很可能你按F2的时候已经跳过去了。
2.bios启动顺序选择USB为第一优先，如果设置了还是直接启动了win7,
那么选择手动选择USB启动，一般是开机F12(每个机型可能有差别，如果不确定百度一下）,然后选择USB(或者有的显示U盘名称)
3.装ubuntu选择与win7共存，否则后期要用EasyBCD进行重建引导分区。

脚本文件：

文件名：oneKeyInstall.sh
#!/bin/bash
apt-get update
apt-get upgrade
apt-get install ubuntu-desktop
ping 127.0.0.1 -c 5
mv 50-ubuntu.conf /usr/share/lightdm/lightdm.conf.d/50-ubuntu.conf
mv 1.profile /root/.profile
ping 127.0.0.1 -c 5
reboot
配置文件-1：1.profile
# ~/.profile: executed by Bourne-compatible login shells.

if [ "$BASH" ]; then
  if [ -f ~/.bashrc ]; then
    . ~/.bashrc
  fi
fi
tty -s && mesg n || true

配置文件-2：50-ubuntu.conf
[Seat:*]
user-session=ubuntu
greeter-show-manual-login=true
allow-guest=false

注意：如果是将这些代码直接写到文件（在win7），一定要保存为unix-ANSI格式，否则win7默认是gbk，复制到linux系统会执行可能会由于编码问题而失败（可以直接下载我的资源文件）

1、通过WinSCP文件传输工具将这三个文件放在服务器的同一个文件夹下




2、putty登录后台执行脚本
bash oneKeyInstall.sh

3、执行结束后在阿里云终端登录（可以用root登录）

一.win7装ubuntu

工具：软通牒
1.把u盘格式化为FAT-32(默认）
2.安装软碟通(网上找注册码注册以后再使用）
如果没有可用如下注册码(亲测可用)：


户名：王涛
注册码：7C81-1689-4046-626F

3.打开要安装的系统

4.写入硬盘映像

5.如果前面没有格式化的在这里也可以选择格式化，已经格式化过的请忽略。（为了一次成功，建议再格式化一次，因为有次只格式了前面一次后来导致安装不成功），这里写入方式选择USB-HDD+(因为是.iso格式)。

6.写入

7.刻录成功（写完之后最好查看下，事件里面数据，介质，扇区之类的是否有写入错误，如果有错误请重新做启动盘，否则安装时进行到一半会出问题）

8.做好启动盘之后就可以安装了.







二： 注意几点：

1.如果安装不成功百度下bios设置问题，我第一次安装就是bios设置错误导致安装不成功，大部分bios设置是开机F2,一定要按开机键之前就按住F2,否则很可能进不去bios,因为电脑自检过程非常快，很可能你按F2的时候已经跳过去了。
2.bios启动顺序选择USB为第一优先，如果设置了还是直接启动了win7,
那么选择手动选择USB启动，一般是开机F12(每个机型可能有差别，如果不确定百度一下）,然后选择USB(或者有的显示U盘名称)
3.装ubuntu选择与win7共存，否则后期要用EasyBCD进行重建引导分区。
